TYDZIEŃ III

Znaczniki HTML:

  • <br> - nowa linia
  • <b> vs <strong> vs <i> vs <em> - elementy do wyróżniania tekstu
  • <b> - pogrubienie teksty, tylko wyróżnienie nie ma znaczenia semantycznego
  • <strong> - pogubienie tekstu, oprócz wyróżnienia ma również znaczenie semantyczne
  • <i> - pochylenie tekstu, kursywa, tylko wyróżnienie nie ma znaczenia semantycznego
  • <em> - pochylenie tesktu, kursywa, oprócz wyróżnienia ma również znaczenie semantyczne np. ironia, może mieć inne znaczenie
  • <a>- linki/ łącza wewnątrz dokumentu
  • <nav> - nawigacja
  • <img> - obrazek o podanym adresie
  • <script> - podłączenie pliku .js

Atrybuty HTML:

  • Id – identyfikuje element (przydaje się m.in. do nawigacji w obrębie dokumentu
  • Class – grupowanie elementów
  • Href – wprowadzamy adres strony lub odnośnika gdzie mamy się przenieść
  • Target – służy do określania miejsca, w którym ma zostać wyświetlona odpowiedź po zatwierdzeniu przez użytkownika formularza lub miejsca otwarcia strony po kliknięciu łącza.
  • Src – source/ źródło
  • Alt – tekst alternatywny, podpis obrazka, który będzie odczytywany np. dla osób niewidomych

JavaScript:

  • consol.log('') - polecenie pokazania co jest napisane
  • ${zmienna} - pozwala na podłożenie zmiennej do treści
  • const zmienna = - definiowanie stałej zmiennej
  • let zmienna = - definiowanie zmiennej którą można później nadpisywać

TYDZIEŃ IV

Znaczniki html:

  • <div> - nie ma znaczenia semantyczne, używamy gdy chcemy coś opakować

CSS:

  • display: block; - elementy domyślnie układają się jeden pod drugim i zajmują całą szerokość
  • display: inline; - elementy domyślnie ustawiają się koło siebie i zajmują tyle szerokości ile same mają
  • display: inline-block; - przyjmują właściwości powyższych i tak ustawiają się obok siebie, ale możemy dowolnie zmieniać ich wysokości i szerokości.
  • width/height - wysokość i szerkość elementu
  • max-width/min-width/max-height/min-height - maksymalny i minimalny rozmiar szerokości i wysokości
  • overflow - pokazuje co ma zrobić element, który się nie mieści w swoim rodzicu
  • overflow: visible; - wartość domyślna
  • overflow: hidden; - wystająca treść jest schowana
  • overflow: auto; - pojawia się scroll dzięki czemu możemy przewijać box
  • padding - wewnętrzne odległości od siebie elementów
  • margin - zewnętrzne odległości od siebie elementów
  • border - ramka elementu
  • border-radius - zaokrąglenie kątów bordera
  • position - pozycjonowanie elementów niezależnie od ustawień domyślnych
  • position: static; - domyślne ustawienie
  • position: relative; - otwiera nam możliwość korzystania z innych ustawień
  • position: absolute; - tylko dokładne usatwienie na stronie
  • position: fixed; - dokładne usatwienie na stronie nawet z przewijaniem strony
  • position: sticky; - dokładne ustawienie w pewnym obszarze, przesuwa się tylko do wyznaczonych miejsc
  • float: left/right; - opływanie tekstu przez inny element z lewej lub prawej strony
  • box-sizing - pokazuje jak wartości w box modelu mają być liczone
  • box-sizing: contetn-box; - domyśle ustawienia
  • box-sizing: border-box; - wysokość i szerkość elementu jest liczona do bordera bez marginu

Jednostki (piszemy zaraz przy liczbach bez spacji):

  • 0 - zero to zero nic więcej nie musimy dopisywać
  • px - piksele (podstawowa jednostka)
  • % - procent, jest brany zawsze od z całości rodzica
  • vw - viewport width, bierze procentową szerokość z całej szerokości strony
  • vh - viewport height, bierze procentową wysokość z całej wysokości strony

JS:

  • typeof - sprawdzenie jakiego typu wartość jest zawarta w zmiennej
  • clear - wyczyszczenie konsoli
  • inner.HTML - ustawia lub pobiera zbiór zawartych w danym elemencie

Typy proste w JS (primitives)

  • String - tekst, otaczamy ' lub " (zalecane ')
  • Number - liczba
  • Bigint - duża liczba
  • boolean - true lub false
  • null - wartość celowo niezdefiniowana
  • undefinded - brak definicji
  • *symbol - tworzy "unikalne opakowanie"

GIT: polecenia w terminalu

  • git config --global user.name -podanie nazwy użytkownika (wpisuje się to tylko raz)
  • git config --global user.email -podanie emaila użytkownika (wpisuje się to tylko raz)
  • git init -założenie repozytorium
  • git config --get user.name -sprawdzenie wpisanych danych
  • git status -informacje o plikach
  • git add . -dodanie plików do zacommitowania
  • git commit -m "opis zmian" -zacommitowanie zmian (dodanie committa)
  • git push -wysłanie zmian do GitHuba
  • gitk -pokazuje historię zmian

CSS:

  • wartość: inherit; - dzieciczenie wartości od rodzica
  • list-style(-type) - styl listy
  • display: flex -
  • flex-direction: row; column; row-reverse; column-revere; - kierunek elementów
  • flex-wrap: nowrap; wrap; wrap-reverse; - zawijanie wiersza/ kolumny
  • flex-flow: row nowrap; <flex-direction>; <flex-wrap>; -
  • justify-content: flex-start; flex-end; centr; space-between; space-around; - ustawienie elementów wzdłuż osi flex-direction
  • align-content: stretch; flex-start; flex-end; center; space-between; space-around; - ustawianie wierszy/ kolumn wzdłuż osi przeciwnej do flex-direction
  • align-items: stretch; flex-start; flex-end; center; baseline; - ustawienie elementów wzdłuż osi przeciwnej do flex-direction
  • align-self: stretch; flex-start; flex-end; center; baseline; - ustawienie elementu wzdłuż osi przeciwnej do flex-direction
  • order: 0; liczba cłkowita - kolejność
  • flex-grow: 0; licba/proporcja;
  • flex-shrink: 1; liczb/proporcja;
  • flex-basis: none; rozmiar bazowy;
  • flex: <flex-grow> <flex-shrink> <felx-basis>
  • background: - tło
  • background-image: url('sciezka-do-pliku'); - zdjęcie jako tło
  • background-position: 0; <pozycja-y>; top; right; center; xx%; xxpx; - ustawienie pozycji tła
  • background-repeat: repeat; repeat-x; repeat-y; no-repeat; round; space; - styl powtarzania obrazka w tle
  • background-attachment: scroll; fixed; local; - jak ma się zachowywać tło
  • background-origin: padding-box; border-box; content-box; -
  • background-clip: padding-box; border-box; content-box; -
  • background-color: wartość koloru; - kolor tła
  • background-image: linear-gradient( color, color) - tło z gradientem
  • background-size: auto; cover; contain; <width> <height>; - rozmiar tła
